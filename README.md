This repository contains my 100 Days of Data Structures and Algorithms (DSA) challenge solved using the C programming language.
The goal is to build strong problem-solving skills and master DSA fundamentals through consistent daily practice.

- Objective
	Strengthen DSA concepts using C
	Improve logical thinking and coding efficiency
	Build consistency with daily problem-solving
	Prepare for coding interviews

- Repository Structure
	100-days-of-dsa/
	│
	├── Day1/
	│   ├── q1.c
	│   ├── q2.c
	│
	├── Day2/
	│   ├── q1.c
	│   ├── q2.c
	│
	├── Day3/
	│   ├── q1.c
	│   ├── q2.c
	│
	...
	│
	└── Day100/
  	  ├── q1.c
  	  ├── q2.cc
  
- File Naming Convention
	Each folder represents a day: Day1, Day2, ..., Day100
	Each problem is stored as:
	q1.c → First question of the day
	q2.c → Second question of the day
